# 9GAG CLONE

A clone version of 9GAG web app.

## Frameworks

Front-End:
 - React - Redux
 - Ant Design

Back-End:
 - Spring Boot
 - Hibernate

Database:
 - Microsoft SQL Server

Services:
 - AWS S3
 - AWS EC2
 - AWS Cloudfront
